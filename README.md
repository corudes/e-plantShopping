e-plantShopping is a React-based web application developed as part of a learning project.  
The application allows users to browse a variety of plants, view plant details, and manage items in a shopping cart.

### Features
- Landing page with a welcoming UI
- Navigation bar with links to:
  - Home (Paradise Nursery)
  - Plants listing
  - Shopping cart
- Add and remove plants from the cart
- Responsive and user-friendly interface

### Technologies Used
- React
- JavaScript (ES6)
- HTML5
- CSS
- Vite

This repository contains the source code and assets for the e-plantShopping project.
